# Environment Variables

## Frontend (.env)

| Variable | Required | Description |
|----------|----------|-------------|
| `VITE_ETHERSCAN_API_KEY` | No | Etherscan API key for faster scanning |
| `VITE_BASE_RPC_URL` | No | Custom Base RPC URL |
| `VITE_WALLETCONNECT_PROJECT_ID` | No | WalletConnect project ID |

## Deployment (.env)

| Variable | Required | Description |
|----------|----------|-------------|
| `PRIVATE_KEY` | Yes | Deployer wallet private key |
| `BASESCAN_API_KEY` | Yes | For contract verification |
